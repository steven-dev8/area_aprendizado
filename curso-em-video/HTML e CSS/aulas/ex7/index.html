Texto grande e pequeno

Para texto ficar grande usamos BIG e pequeno usamos small

Texto deletado

Para deletar o texto usamos a tag DEL, dessa forma.

Texto inserido

Para deixar o texto inserido usamos o INS

Essa daqui é outra forma de inserir texto, porém não é semântica

Texto sobrescrito

Para texto sobrescrito usamos a tag SUP, ex: n2 = 2

Texto subscrito

Para texto subscrito usamos a tag SUB como no exemplo a seguir. Ex: StevenX99
