Vamos falar de citações e código.

Tag "code"

Esse é meu código normal usando apenas a tag P, cada caracter tem um tipo de espaço diferente

E esse é um texto usando a formatação CODE que também é uma tag, ele exibe o texto com uma fonte monoespaçada, agora cada caracter tem o mesmo tamanho.

Tag "pre"

        
        number = int(input('Número: '))
        print(f'Seu número é {number}')

A tag PRE visualiza o texto exatamente como no código, contando os espaços entre as coluna e as quebra de linha. O exemplo acima é usando PRE e CODE. ↑

Tag "q"

A tag "q" marca uma citação simples.

Ex: Minha mãe uma vez disse: você não é todo mundo

Tag "blockquote"

A tag "blockquote" faz uma citação completa.

Na bíblia, Gênesis capítulo 1 e versículo 1 diz o seguinte:

No princípio criou Deus o céu e a terra.
