Tag "a" e atributo "href" "target" e "rel"

A tag "a" é usada para adicionar um HyperLink
O atributo "href" é usado dentro da tag "a" para colocar o link onde o HyperLink será direcionado
O atributo "target" é usado para especificar se o site do HyperLink será aberto na mesma aba ou em outra.
O atributo "rel" é usado para configurações de mecanismo de busca, como otimização, indicar que o link é externo e segurança
