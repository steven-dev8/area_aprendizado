Listas ordenas e Não ordenadas

Tag "ol"

ol = Order list, cria uma lista ordenada. Você pode usar também o parâmetro "type"
type="1 A a I i"

  1. Acordar
  2. Banhar
  3. Escovar os dentes
  4. Tomar café

Tag "dl" "dt" e "dd"

São tags utilizadas para fazer uma lista de definição, isto é, colocar o termo e sua definição. Primeiro abre a lista com a tag "dl" e em seguida coloca o termo na tag "dt" e na outra linha colocar a definição na tag "dd" Ex:

a = input()
Dá uma entrada de dados string para a variável "a"
while
É uma estrutura condicional de repetição, o codigo identado ira rodar até a condição do while ser verdadeira.
for
É uma estrutura de repetição limitada, o codigo identado ira rodar pela quantidade de vezes especificada dentro do range()
